ComplaintPost
=============

Schema
------

:title:
   string, required

:description:
   string, required

:documents:
   List of :ref:`document` objects

:relatedPost:
    string

    Id of related :ref:`complaintpost`.

:recipient:
    string, required

    Possible values are:

    * `complaint_owner`
    * `tender_owner`
    * `aboveThresholdReviewers`

:author:
    string, autogenerated

    Possible values are:

    * `complaint_owner`
    * `tender_owner`
    * `aboveThresholdReviewers`

:datePublished:
   string, :ref:`date`, autogenerated
